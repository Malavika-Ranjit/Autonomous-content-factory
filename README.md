ContentIQ AI
Project Description

Project Description

ContentIQ AI is a web-based application that automates content creation. It allows users to generate blogs, social media posts, and emails instantly by providing a topic or prompt. The tool also includes a “Highlight Key Points” feature that extracts the main ideas from any generated or uploaded text. This helps users quickly understand and summarize content, making writing and editing more efficient.

This project addresses the problem of time-consuming manual content creation and content summarization, enabling users to produce high-quality, ready-to-use content in minutes.

Features
1.Blog Generation – Enter a topic and automatically get a detailed, well-structured blog post.
2.Social Media Post Generation – Generate catchy and engaging posts for platforms like Facebook, Instagram, and Twitter.
3.Email Generation – Create professional emails quickly based on the user’s prompt or topic.
4.Highlight Key Points – Automatically extract the main points from generated content or uploaded text/PDFs.
5.PDF Text Extraction (Optional) – Upload PDF files to extract text and use it as input for content generation.
6.Real-time Content Generation – Instant AI-generated results without needing a backend server.
7.Simple and Responsive UI – Minimal design using React.js and CSS for easy use.

How It Works
1.User enters a topic, prompt, or uploads a PDF.
2.User selects the content type: Blog, Social Post, or Email.
3.The app sends the input to the backend API (OpenAI / Groq API) for content processing.
4.The backend AI API generates high-quality content based on the user’s input.
5.The generated content is returned to the frontend and displayed instantly in the UI.
6.Generated content or highlighted key points are returned to the frontend and displayed instantly.
7.Users can copy or use the content directly.

Tech Stack
Frontend
React.js – For building interactive UI and components.
CSS – For styling the application.
React Router – For navigation between different pages or content types.

Backend / API
OpenAI API / Groq API – Handles AI-powered content generation and key point extraction.
PDF.js – Extracts text from uploaded PDFs for content generation or summarization.

Other Tools
